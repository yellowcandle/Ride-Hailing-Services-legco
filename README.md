# Ride-Hailing-Services-legco

香港立法會「與規管網約車服務有關的4項附屬法例小組委員會」（sc103）公眾意見書存檔及分析。

Archive and analysis of public submissions to the HK LegCo Subcommittee on Four Pieces of Subsidiary Legislation Relating to Regulation of Ride-Hailing Services.

## 分析報告 / Analysis report

**https://yellowcandle.github.io/Ride-Hailing-Services-legco/**

## Contents

- `submissions/*.pdf` — 16 public submission PDFs (snapshot 2026-06-11; 43 paper entries, ~430 individual submissions)
- `submissions/INDEX.md` — manifest mapping paper numbers to files
- `submissions/md/` — markitdown text extractions
- `submissions/ANALYSIS.md` / `ANALYSIS.html` — stance tallies, party grouping, stakeholder background research
- `docs/index.html` — published report (GitHub Pages)
